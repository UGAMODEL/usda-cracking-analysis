# USDA Cracking Analysis

This repository contains data and scripts for analyzing pecan cracking outcomes from a full-factorial experiment of meyer cracker gathered in july 2024. 
The overarching goal is to understand how factors such as **motor speed**, **spring stiffness**, **displacement setting**, and **moisture level** influence different crack classifications.
